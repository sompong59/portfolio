ລະບົບບັນທຶກເລກທີ

HTML ເປັນພາສາມາດຕະຖານສຳລັບການສ້າງໜ້າເວັບ.

ດ້ວຍ HTML ເຈົ້າສາມາດສ້າງເວັບໄຊຂອງເຈົ້າເອງໄດ້.

ເລີ່ມຮຽນ HTML ດຽວນີ້!

ຕົວຢ່າງ HTML

ຢ່າກັງວົນຖ້າເຈົ້າບໍ່ເຂົ້າໃຈຕົວຢ່າງນີ້ເທື່ອ.

ເຈົ້າຈະໄດ້ຮຽນຮູ້ກ່ຽວກັບແທັກ HTML ທັງໝົດໃນພາຍຫຼັງ.


<!DOCTYPE html>

<html>

<head>

<title>ໜ້າທຳອິດຂອງຂ້ອຍ</title>

</head>

<body>


<h1>ນີ້ແມ່ນຫົວຂໍ້ໃຫຍ່</h1>

<p>ນີ້ແມ່ນຂໍ້ຄວາມ.</p>


</body>

</html>
